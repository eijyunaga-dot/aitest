**AI比較アプリケーション v1.0**
- Python 3.8以上
- メモリ: 8GB以上推奨
- ディスプレイ: 1200px以上の幅推奨
- このプロジェクトは個人使用を目的としています。
- 利用規約遵守のため自動入力等は未実装です

**AI比較アプリケーション v1.0**
*インストーラー版*

【インストール方法】
1. AI比較アプリ_Setup_v1.0.exe をダブルクリック
2. インストールウィザードに従う
3. 「デスクトップアイコンを作成」をチェック推奨
4. インストール完了

【起動方法】
- デスクトップアイコンから起動
- またはスタートメニュー > AI比較アプリケーション

【アンインストール】
- コントロールパネル > プログラムのアンインストール
- または スタートメニュー > アンインストール

【動作環境】
- Windows 10/11 (64bit)
- メモリ: 8GB以上推奨
- 画面解像度: 1366x768以上を想定

【注意事項】
- 初回起動時に各AIサービスでログインが必要(各窓毎にアカウント情報別保存)
- Windows Defenderが警告を表示する場合があります
  → 「詳細情報」→「実行」で起動できます
